# jerelmoffatt.github.io
Hello world pages example

The file `index.html` is what is displayed when anyone visits https://jerelmoffatt.github.io

This `index.html` page can access other files from the following places
css/         <-- Contains styling information for the html that is contained in index.html
js/          <-- Contains any javascript 
pics/        <-- Contains any pictures you want your website to display or link to
docs/        <-- Contains any documents (pdfs, docs, etc.) 

And of course, your website can provide links to any other web page (or area of github) you want it to.
